# 商業規則規格

## 條碼（Barcode）

### 格式定義
- 長度：**13 位數字**
- 結構：`YYY` + `MM` + `DD` + `SSSSSS`
  - `YYY` = 民國年（3碼，例如 115）
  - `MM`  = 月份（2碼，01–12）
  - `DD`  = 日期（2碼，01–31）
  - `SSSSSS` = 當日流水號（6碼，000001 起）
- 範例：`1150417000205` = 民國115年4月17日第205號

### 驗證規則
1. 必須為純數字（`/^\d+$/`）
2. 長度必須等於 13
3. 警告（非強制拒絕）：前7碼應可解析為有效日期
4. 空白輸入 → 忽略，不加入記錄

### 重複條碼
- 預設允許：同一條碼可多次出現（代表多件相同貨品）
- 若需去重，需在需求中明確說明

## 品號（Pinghao）

### 格式定義
- 格式：`A00016`（1個大寫英文字母 + 5個數字）
- 輸入時自動轉為大寫
- 若未填入，顯示為 `(未設定)` 但仍記錄

### 驗證規則
1. 允許空白（顯示「未設定」，仍可繼續操作）
2. 建議格式：`/^[A-Z]\d{5}$/`，不符時顯示黃色警告框，但不阻止儲存

---

## 數量（Qty）

- 預設值：**1**
- 允許範圍：1 – 999
- 型別：正整數
- 無效值（0、負數、非數字）→ 回退為 1

---

## 掃描行為

1. 掃描槍送出條碼後接 Enter（或 Tab）
2. 系統捕捉 Enter/Tab → 取輸入框內容 → 驗證 → 加入記錄
3. 加入成功後清空輸入框，焦點留在掃描框
4. 品號和數量設定**不因掃描而清除**（批次掃描同品項）

---

## 匯出規則

### Excel 格式（.xlsx）
| 欄 | 內容 | 寬度 |
|----|------|------|
| A  | 條碼（barcode） | 18 |
| B  | 品號（pinghao） | 10 |
| C  | 數量（qty）     | 8  |

- **無標題列**（第一行即資料）
- 工作表名稱：`工作表1`
- 檔名：`出貨件數_民國年.月.日.xlsx`（例如 `出貨件數_115.04.18.xlsx`）
- 排序：依掃描時間先後（不排序）

### 民國年計算
```
民國年 = 西元年 - 1911
```

### TSV 格式（貼到 Excel）
```
{barcode}\t{pinghao}\t{qty}
```
每筆以 `\n` 分隔，無標題列。

---

## 統計規則

- 依品號分組，顯示：筆數（記錄數）、總數量（qty 加總）
- 品號排序：字母順序（A-Z）
- 合計列顯示總筆數與總數量
